# CV Creator

Create your CV/Resume with this simple tool developed using React, TypeScript, Vite, SCSS and ReactPDF.
[CV Creator](https://cv-creator-ten.vercel.app/)

## Running with npm

Open your terminal on the repo directory and run

```bash
> npm install
```

Then...

```bash
> npm run dev
```
